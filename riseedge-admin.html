How To Use
Complete guide to managing your RiseEdge website.
📌 Adding a New Paid Spotlight Author
1Author pays reservation fee → you receive payment
2Go to Spotlight → click Add New Author
3Upload their photo, fill in name, bio, genre, event date & Amazon link
4Toggle Set as Featured if this is the next upcoming event
5Click Save — profile goes live on website instantly and permanently
💾 Why data is now permanent

Data is saved to localStorage in your browser. Since both the admin panel and main website are on the same Vercel domain (rise-edge-book-club.vercel.app), they share the same storage — so all changes are permanent and instantly visible on the main website.

🔑 Change your password

Open riseedge-admin.html in a code editor. Search for riseedge and admin2024 and replace with your username and password. Then re-deploy to Vercel.
